### 💪 Fitty — PPL Workout Tracker

A full-stack **Push-Pull-Legs workout tracker** built around consistency and real progress visualization — because spreadsheets aren't a training plan.

**Stack at a glance:**
- ⚡ **React 18 + Vite** — snappy, instant UI  
- 🐻 **Zustand** — clean global state  
- 📊 **Recharts** — progress you can actually see  
- ☁️ **Supabase** — real-time sync + auth  
- 📱 **PWA** — offline-first, installable  
- 🎨 **Tailwind CSS** — dark-mode by default
